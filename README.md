<h3 align="center">3DSpotify | Playback controller</h3>

<p align="center">
Spotify API controller software for 3DS
 <br>
 <a href="https://ssambender.github.io/3DSpotify-Install/"><strong>Install Instructions »</strong></a>
</p>

___

### Features
- Play/pause music
- Skip/rewind music
- View and select playlists to play from
- Display song, artist, and album info
- Display current timestamp
- QR code scanner for auth
